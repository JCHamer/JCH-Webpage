Settlers of Catan Board

Settlers of Catan - the custom designed and lasercut board.

At the end of my Linear Algebra Class, my team of four decided to explore Algorithmic Music Generation. We studied the application of Markov Chains to music generation and mapped a sequential progression using the weights from a transition matrix as well as previous notes in order to generate music. Most of the code we wrote can be found here.

We decided to use Yankee Doodle to calculate our transition weights. I wrote a quick python script that read in all the notes and their durations, counted the number of appearances of each note, and output a table of the number of occurences of each note relative to the total number of notes. This was used that to build a basic 0-order transition matrix. Next, I counted the occurence of each two-note permutation to build the first order matrix that gave a weight for the next note based on the note that was just played. Following this, I counted three note sequences to build the second order matrix in a similar manner. Additionally, I followed the same process to create transition matrices for each note duration that shows up in Yankee Doodle - quarter, eighth, dotted eighth and sixteenth notes. After I had produced the transition matrices, a friend wrote a MATLAB script to simulate the sequence following my weights and produced a seqeunce of notes and their corresponding durations. Afterwards I formatted the results and ran them through PySynth, a music synthesizer written in Python, which outputs a .wav file that sounds like a Piano playing the music that we input. I've run the Hour of Code for over 250 students! While preparing for the most recent Hour of Code I worked with a teacher to look at how we could provide High School students who already had experience in Computer Science a more challenging experience in order for them to still be able to learn something new. We settled on teaching the students some basic Python skills. As I was going to be running the event for an entire classroom at a time, I decided to make a cheat-sheet for the students to solve problems without needing my immediate help. I decided that there is no better way to make a cheat-sheet for Python than by using Python, so I wrote a script to walk an inexperienced Programmer through the basics of Python! After I had taught the tools needed for basic math, I brought in a friend who suggested using turtle, a python learning environment that lets a student use simple python skills to draw (with a turtle). Using my initial framework, we expanded my Python Tutor to also walk a student through their first steps in Python!

Mechanical Keyboard Project

My family plays Settlers of Catan, no really, we play a lot of Catan. Over the years, our original set has gotten a lot of love and has become a little bit warped after one too many children spilled apple cider on it (apparently it only takes one) and we eventually had to buy a new set. After moving to College, I wanted to take a game with me and was offered the old set, but I had no good hexes to use. So I spent the beginning of my summer designing a set of hexes based on the numerous sets of laser cut hexes at Thingiverse. Having started with the beautiful hex drawings, I learned how to use inkscape, a vector drawing program and drew all the lines to cut out my hexes and removed all the extra lines so nothing was cut twice. Then I set about making the extra hexes that I didn't find interesting looking versions, for example, I created the Ore hex!

The next step was to actually go and print out the hexes! Thanks to all the hours of work behind the computer, I now get to pick out wood, cut it down, and sit back and watch the laser burn it's way across (to the left) and then through (to the right) my sheet of wood. After many, many test cuts, I was able to get the burns consistently easy-to-read. Until eventually we could play Catan!
