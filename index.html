Settlers of Catan Board

A custom Settlers of Catan board that I designed and laser printed.

At the end of my Linear Algebra Class, my team of four decided to explore Algorithmic Music Generation. We studied the application of Markov Chains to music generation and mapped a sequential progression using the weights from the stochastic transition matrix as well as the previous state of the output in order to generate a sequence of musical notes and durations. Most of the code we wrote can be found here.

We decided to use Yankee Doodle to build our transition weights. I wrote a quick python script and read in all the notes and their durations, and then counted the number of appearances of 'g's out of the total notes and used that to build a naive 0-order transition matrix. I then counted the frequency of all the permutations of two note sequences to build the first order matrix that used a state value to determine the next note weights based on the note that was previously played. Following this, the count was a three note sequence to build the second order matrix in a similar manner. I also followed the exact same process to create transition matrices for each note duration that shows up in Yankee Doodle - quarter, eighth, dotted eighth and sixteenth note. After I had produced the transition matrices, a friend wrote a MATLAB script to simulate the sequence following my weights to produce a seqeunce of notes and corresponding durations. Afterwards I formatted the results and ran them through PySynth, a music synthesizer written in Python, which output a .wav file that sounded like a Piano playing the music that was input.

I've run the Hour of Code for over 250 students! While preparing for the most recent Hour of Code I worked with a teacher to look at how we could provide High School students with a stronger background in Computer Science a more challenging experience where they would still be able to learn something new. After going back and forth, we settled on teaching the students some basic Python skills. Because I was going to be running the experience for an entire classroom at a time, I decided to look for a cheat-sheet for the students so that they could solve problems without needing my immediate help. After my search turned up nothing that was exactly what I wanted, I decided to make my own. I decided that there is no better way to make a cheat-sheet for Python than by using Python to make a cheat-sheet. I wrote a script to walk an inexperienced Programmer through the basics of Python! After I had in basic math functionality, I brought in a friend who helped suggested using turtle, a python learning environment that lets a student use basic python skills to draw on their screen. Using my initial framework, we built a Python Tutor that walks a student through their first steps in Python!
